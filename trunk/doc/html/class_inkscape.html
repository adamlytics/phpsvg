Definition at line 416 of file inkscape.php.

exportAreaSnap ( )

For PNG export, snap the export area outwards to the nearest integer SVG user unit (px) values. If you are using the default export resolution of 90 dpi and your graphics are pixel-snapped to minimize antialiasing, this switch allows you to preserve this alignment even if you are exporting some object's bounding box (with --export-id or --export-area-drawing) which is itself not pixel-aligned.

setExportId ( $  id,
$  exportIdOnly = false 
)

Set the ID of the object whose dimensions are queried. If not set, query options will return the dimensions of the drawing (i.e. all document objects), not the page or viewbox

Only export to PNG the object whose id is given in --export-id. All other objects are hidden and won't show in export even if they overlay the exported object. Without --export-id, this option is ignored

Parameters:
string$id
booleanAll other objects are hidden and won't show in export even if they overlay the exported object.
